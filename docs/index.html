Installation

Prerequisites

  1. Download Android Studio

    • Have a device in AVG manager set up
    • I am currently using a Pixel 2

  2. Have Node.js Installed

  3. Install Git (Group Use)

Can skip steps 1-3 and 5-6 if you clone the repository

  1. Create project
    ionic start "project name" tabs --type=react
  • "project name" being the project title
  • Uses tabs interface

  1. Enter folder of newly created project

    cd xplore

  2. Run serve

    ionic serve

  3. Build project

    npm run build

  4. Generate a new capacitor project to allow emu to work

    npx cap init

  5. Add devices appropriately 

    npx cap add ios

    or

    npx cap add android

NOTE: ios emulator devices will only work on ios devices with xcode

  1. Sync capacitors to devices

    npx cap sync ios

    or

    npx cap sync android

  2. Open project on xcode or Android Studio

    npx open ios

    or 

    npx open Android
  • If you get an error, run this command instead:
    npx open <device name>
  1. Run emulator with live server to reduce redundancy
    ionic capacitor run ios --livereload
    or
    ionic capacitor run android --livereload
  • Here you will choose your device and will direct you to a localhost server.
  • If an error occurs while trying to connect to hose, run this command instead:
    ionic cap run ios -l --external
    or 
    ionic cap run android -l --external
    NOTE: Instead of rebuilding projects and copying files over to xcode or android studio, we can have a live server to our emulators.

At its core lies the real-world, interactive treasure hunting application that you will develop over the course of this semester. The resulting treasure hunting experience will provide users with a new means for interacting with local businesses and community members.
